# 3D Layout Designer Images

Place your panoramic 360° images here.

## Default Image

The default panoramic image should be named `panorama.jpg` (or `.png`, `.JPG`, etc.)

Supported formats: JPG, PNG, WebP

## How to Add Your Image

1. Save your panoramic image to this directory
2. Name it `panorama.jpg` (or update the path in the code)
3. The image will automatically load when you open the 3D Layout Designer

## Image Requirements

- 360° equirectangular panoramic format
- Recommended resolution: 4096x2048 or higher
- Aspect ratio: 2:1 (width:height)
